# 1NCE IoT SIM Configuration

## Overview
- **Provider**: [1nce.com](https://1nce.com)
- **Plan**: Lifetime — 500 MB / 10 years per SIM
- **Unit cost**: ~$14 (100-pack)
- **Coverage**: 170+ countries

## APN Settings
| Field | Value |
|---|---|
| APN | iot.1nce.net |
| Username | (leave blank) |
| Password | (leave blank) |
| Auth Type | None / PAP |

## SIM Activation
1. Log in to [portal.1nce.com](https://portal.1nce.com)
2. Navigate to **SIM Cards** → **Activate**
3. Enter ICCID (printed on SIM card)
4. Assign to device in tracker config

## Data Budget per SIM (7-Lever Optimised)
| Scenario | Monthly Usage |
|---|---|
| All levers ON | ~0.3 MB |
| Levers 1–4 only | ~0.7 MB |
| No optimisation | ~4–8 MB |

> At 0.3–1.5 MB/mo, 500 MB lasts 28–139 years — effectively lifetime.
