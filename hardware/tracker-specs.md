# GPS Tracker Specs & Sourcing

## Recommended Model
- **Type**: OBD2 GPS 4G (LTE Cat-1 or Cat-M1)
- **Source**: Alibaba — search `OBD2 GPS 4G no monthly fee`
- **Unit cost**: ~$25 (MOQ 100)

## Key Requirements
- 4G LTE connectivity
- OBD2 plug-and-play (no wiring)
- IP67 weather resistance
- Internal battery backup
- Supported protocols: GT06, H02, Teltonika, or similar (Traccar-compatible)
- Ports: 5055–5140 (Traccar default range)

## Verified Compatible Protocols (Traccar)
| Protocol | Port |
|---|---|
| GT06 | 5023 |
| H02 | 5013 |
| Teltonika | 5027 |
| TK103 | 5002 |

> Confirm with supplier which protocol the device uses before ordering.
